# 档案管理系统数据库表设计

## 1.用户表 user

| 字段名      | 类型         | 默认值 | 备注                                    | 约束       |
| ----------- | ------------ | ------ | --------------------------------------- | ---------- |
| id          | int unsigned | 无     | 用户id                                  | 主键、自增 |
| username    | varchar(50)  | 无     | 用户名                                  | 唯一       |
| name        | varchar(50)  | 无     | 姓名                                    |            |
| password    | varchar(50)  | 123456 | 密码                                    |            |
| role        | tinyint      | 无     | 角色, 说明: 1 浏览者,2 上传者, 3 管理员 |            |
| gender      | tinyint      | 无     | 性别 , 1 男 ; 2 女                      |            |
| create_time | datetime     | 无     | 创建时间                                |            |
| update_time | datetime     | 无     | 更新时间                                |            |



## 2.文件表doc

| 字段名        | 类型         | 默认值 | 备注           | 约束       |
| ------------- | ------------ | ------ | -------------- | ---------- |
| id            | int unsigned | 无     | 文件id         | 主键、自增 |
| filename      | varchar(100) | 无     | 文件名         | 唯一       |
| operator_name | varchar(50)  | 无     | 上传者姓名     |            |
| operator_id   | int unsigned | 无     | 上传者id       |            |
| upload_time   | datetime     | 无     | 上传时间       |            |
| url           | varchar(100) | 无     | 文件的访问路径 |            |
| uuid          | varchar(100) | 无     | 文件的UUID     |            |



## 3.操作日志表operate_log

| 字段名        | 类型          | 默认值 | 备注                   | 约束       |
| ------------- | ------------- | ------ | ---------------------- | ---------- |
| id            | int unsigned  | 无     | ID                     | 主键、自增 |
| operate_user  | int unsigned  | 无     | 操作人ID               |            |
| operate_time  | datetime      | 无     | 操作时间               |            |
| class_name    | varchar(100)  | 无     | 操作的类名             |            |
| method_name   | varchar(100)  | 无     | 操作的方法名           |            |
| method_params | varchar(1000) | 无     | 方法参数               |            |
| return_value  | varchar(2000) | 无     | 返回值                 |            |
| cost_time     | bigint        | 无     | 方法执行耗时，单位：ms |            |
